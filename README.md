# odin-recipes
Repository for the Foundations HTML Project: https://www.theodinproject.com/lessons/foundations-recipes

## Overview
This is the first project for the Odin Project. It is the capstone project for the introductory HTML section. I have learned most of this in the past but it was a good refresher.

## Deviations
Inspired by the images on the Serious Eats recipe which have a "Credit:" line underneath the photo. I researched and added the `<figure>` elements to each recipe page which allows for a caption in `<figcaption>`
